Scene Setters

HTML, CSS, and JavaScript

Scene Setters is a film equipment rental company I created from scratch. I designed the logo and designed the UI/UX for the site. Here is the wireframe I created using Figma. I did the necessary research to discover my target audience and created features based on audience needs.

This site has an equipment section with filter buttons for the prospective items. It also has a packages section where customers are able to see which package aligns with their specific needs and budget. Lastly at the bottom there is a contact form to contact the company.

Overall, this was a fun project to create because I was able to exercise my creativity. The features I enjoyed creating have to be the packages section with the hover feature, the filtered equipment section, and the back to the top arrow. The main struggle I've encountered was getting rid of the blue text for phone numbers and the "see packages" button in mobile view.

Be sure to check out the site and the GitHub repository to see it in action.
